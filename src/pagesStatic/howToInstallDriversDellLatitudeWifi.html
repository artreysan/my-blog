How to Install Drivers WiFi Dell Latitude E6420

Search our card of wireless


        sudo dmesg

        sudo dmesg | grep -i b43

Finding out your chipset / wireless card name on a Arch Linux


        sudo lspci

        sudo lspci | grep -i broadcom

If you card chipset is broadcom install


        sudo pacman -S broadcom-wl-dkms

Info: https://archlinux.org/packages/community/x86_64/broadcom-wl-dkms/

Then reboot the system


        sudo reboot

Verify and connect to the Wireless network


        ip a s 

        sudo pacman -S lshw

        sudo lshw -C network -short

        nmtui
