Create proyect React with Vite.js

Execute the follow command:


        npm create vite

Write the name proyect:


        ? Project name: »

Select framework:


        ? Select a framework: » - Use arrow-keys. Return to submit.
            vanilla
            vue
        >   react
            preact
            lit
            svelte

Select variant from javascript or typescript


        ? Select a variant: » - Use arrow-keys. Return to submit.
        >   react
            react-ts

Exit: 


        √ Project name: ... my-blog
        √ Select a framework: » react
        √ Select a variant: » react

        Scaffolding project in C:\Users\arturo\workspace\proyects\my-blog...

        Done. Now run:

        cd my-blog
        npm install
        npm run dev

Next install dependecies:


        npm i

Verify the installation:

npm run dev
