Install BSPWM + Polybar ArchLinux

Package for configuration

sudo pacman -S nitrogen git dmenu

Install bspwm and dependences

sudo pacman -S bspwm sxhkd

Configuration BSPWM


    mkdir ~/.config/bspwm

    cp /usr/share/doc/bspwm/examples/bspwmrc ~/.config/bspwm/

Configure SXHKD


    mkdir ~/.config/sxhkd

    cp /usr/share/doc/bspwm/examples/sxhkdrc ~/.config/sxhkd/

Edit the file ~/.config/sxhkdrc

Terminal configartion


    ...        
    # terminal emulator
    super + Return
            alacritty
    ...

App Finder


    ... 
    # program launcher
    super + D 
            dmenu_run
    ...

Install Polybar

sudo pacman -S polybar

Configuration


    mkdir ~/.config/polybar
    cp /usr/share/doc/polybar/examples/config.ini ~/.config/polybar/

Create and write in file ~/.config/polybar/launch.sh


    #!/usr/bin/env bash

    # Terminate already running bar instances
    # If all your bars have ipc enabled, you can use 
    polybar-msg cmd quit
    # Otherwise you can use the nuclear option:
    # killall -q polybar

    # Launch bar1 and bar2
    echo "---" | tee -a /tmp/polybar1.log /tmp/polybar2.log
    polybar example 2>&1 | tee -a /tmp/polybar1.log & disown
    #polybar bar2 2>&1 | tee -a /tmp/polybar2.log & disown

    echo "Bars launched..."

Make it executable:


    chmod +x $HOME/.config/polybar/launch.sh

Add the following line to ~/.config/bspwm/bspwmrc


    $HOME/.config/polybar/launch.sh
