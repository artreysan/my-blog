Adding Remote in Repository Existing

  • 1. Create repository
  • 2. Copy url of repository: example ===> https://github.com/beartree-ars/prueba-git02.git

Inside in Proyect

Created file "README.md"

touch README.md

In folder existing execute:

git init

Add all files in Staging Area with

add .

Add commit:

git commit -m "----Message----"

Change branch name to MAIN: 

git branch -M main

Add Remote to repository https://github.com/beartree-ars/prueba-git02.git

git remote add origin https://github.com/beartree-ars/prueba-git02.git

Send the files to Remote Repository

git push -u origin main
