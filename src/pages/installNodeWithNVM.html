Installed NodeJS with NVM (Node Version Manager)

For install NodeJS in Linux we need to have installed, CURL and BASH

pacman -S curl bash

After execute the follow command:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash

Update the configuration file in ".bashrc" with

source ~/.bashrc

Or if you have Oh-My-ZSH copy and paste the follow text in ".zshrc":


        export NVM_DIR="$HOME/.nvm"
        [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"  # This loads nvm
        [ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"  # This loads nvm bash_completion

After execute the follow command: source ~/.zshrc
